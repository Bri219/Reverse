The Code for REVERSE

                        
                            // Get the String from the page
                            // Controller Function
                            function getValue(){
                            
                              document.getElementById("alert").classList.add("invisible");
                              let userString = document.getElementById("userString").value;
                              let revString = reverseString(userString);
                              displayString(revString);
                            }
                            
                            
                            // Reverse the String
                            // Login Function
                            function reverseString(userString){
                            
                              let revString = [];
                              // Reverse a string using a for loop
                              for (let index = userString.length - 1; index >= 0; index--) {
                                   revString += userString[index];
                              }
                              return revString;
                            }
                            
                            
                            // Display the reversed String to the user
                            // View Function
                            function displayString(revstring){
                            
                              //write to page
                              document.getElementById("msg").innerHTML = `Your string reversed is: ${revstring}`;
                              //show the alert box
                              document.getElementById("alert").classList.remove("invisible");
                            }

The Code is structered in three functions

getValue

The first function, getValue(), is the controller function that gets the string from the page. It does this by accessing the value of the element with the id of “userString” and storing it in a variable called userString. Then, it calls another function, reverseString(userString), passing it the user’s input as an argument. This function returns the reversed version of the user’s string, which is then stored in another variable called revString. Finally, the controller function calls a third function,displayString(revString) , passing it the reversed string as an argument. This function displays the result to the user on the page.


reverseString

The second function, reverseString(userString), is the logic function that reverses the string. It does this by creating an empty array called revString and using a for loop to iterate over each character of the user’s string in reverse order. For each iteration, it appends the current character to the end of the revString array, effectively reversing the order of the characters. Then, it returns the revString array as a single string.


displayString

The third function, displayString(revstring), is the view function that displays the reversed string to the user. It does this by writing to the element with the id of “msg” and inserting the reversed string into a template literal that says “Your string reversed is: …”. Then, it removes the “invisible” class from the element with the id of “alert”, which makes it visible on the page. This way, the user can see their input and output.
